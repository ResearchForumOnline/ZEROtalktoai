# Security Policy

Please report security concerns through the contact route published on the live TalkToAI site:

[https://talktoai.org/](https://talktoai.org/)

Do not publish exploit details, credentials, private keys, API tokens, admin screenshots, server paths, customer data, or private implementation notes in GitHub issues.

This repository is a public project index only. Production code, infrastructure automation, and protected operational details are not included here.
