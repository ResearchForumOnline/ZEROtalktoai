# Contributing

Thank you for helping improve the TalkToAI public project hub.

This repository is for public project navigation, documentation, and portfolio structure. Keep contributions focused on information that can safely be public.

## Good Contributions

Useful contributions include:

- Fixing broken public links.
- Improving project summaries and public documentation.
- Suggesting clearer repository descriptions or GitHub profile structure.
- Improving security guidance around public documentation boundaries.
- Suggesting public ZSEC security advisory sources or Linux hardening references.

## Do Not Include

Do not submit:

- Passwords, SSH keys, API keys, tokens, cookies, or private credentials.
- Server IP/login operations notes that are not already meant for public documentation.
- Customer data, private emails, invoices, contracts, or internal migration notes.
- Working exploit chains against live systems.
- Private source code that has not been approved for public release.

## Pull Request Style

For documentation changes:

1. Keep the wording direct and practical.
2. Prefer public links over private references.
3. Explain why the change helps visitors understand the project.
4. Keep security-sensitive details private.

## Security Reports

For security-sensitive issues, read `SECURITY.md` first and avoid posting sensitive details publicly.
